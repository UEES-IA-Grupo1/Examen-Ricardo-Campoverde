# Examen Práctico – Inteligencia Artificial

Repositorio correspondiente al examen práctico de la asignatura **Inteligencia Artificial**.

## Contexto
El proyecto adapta un sistema de análisis de sentimientos para evaluar **reseñas hoteleras**, incorporando vocabulario específico del sector turismo.

## Funcionalidades implementadas
- Diccionario de palabras positivas y negativas orientado a hoteles
- Emoticones personalizados por tipo de sentimiento
- Mensajes de salida adaptados a reseñas hoteleras
- Nuevo criterio de neutralidad basado en diferencia ≤ 1
- Casos de prueba con reseñas reales de hoteles

- `Examen_Campoverde_IA_12_2025.ipynb`: Notebook con el desarrollo completo y ejecución exitosa
